# Softwares required to use source codes
- Python
- FFMPEG
If you are going to use the source code, ffmpeg.exe must be present in your project directory.
You can download FFMPEG from [https://www.gyan.dev/ffmpeg/builds/ffmpeg-release-full.7z](gyan.dev/ffmpeg). After downloading, go to the bin folder inside the compressed file and copy all 3 files there to your project directory.
